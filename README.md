# GDorisPhotoKit
## Github上的相册资源相关的开源库大多数代码质量不高，目的是构建一个强大的相册选择，浏览，编辑的代码库。
目前代码还在alpha Test阶段，一些功能还在实现阶段。大部分功能均没经过单元测试。不可用于成熟的项目中。
